# L19 - Projekt 2 (del I)

## Dagordning
* Repetition av neurala nätverk - träning för hand med mera.
* Planering och/eller påbörjande av **Projekt 2** - Implementering av ett neuralt nätverk i ett inbyggt system.

## Mål med lektionen
* Ha fräschat upp i minnet vad ett neuralt nätverk är och hur de tränas.
* Ha valt appliktion för **Projekt 2**, exempelvis:
    * Ett neuralt nätverk som avgör om en spelare vinner eller förlorar i ett enkelt spel baserat på knappsekvenser.
    * Ett system som klassificerar olika knappmönster eller sensordata och styr flera lysdioder. 
    * Ett enkelt ”smart hem”-scenario där nätverket avgör om en lampa ska tändas baserat på flera insignaler (knappar, timers, sensorer).
    * Ett musik- eller rytmrelaterat projekt där knapptryckningar tolkas och klassificeras. 
    * Given defaultuppgift.
